# Tic-Tac-Toe

A simple client-server Tic-Tac-Toe game implemented in Haskell.

## Getting Started

### Prerequisites

*   [stack](https://docs.haskellstack.org/en/stable/README/)

### Building

```bash
stack build
```

### Running the Server

```bash
stack exec server
```

### Running the Client

```bash
stack exec client
```

## Running the Tests

```bash
stack test
```
